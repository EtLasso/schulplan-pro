# SchulPlan Pro

Moderne Demo-Anwendung fuer digitalen Schulalltag mit Stundenplan, Aufgaben, Abwesenheiten, Ressourcenverwaltung und einer responsiven Benutzeroberflaeche.

## Live Demo

[SchulPlan Pro im Browser starten](https://etlasso.github.io/schulplan-pro/)

## Funktionen

- Dashboard mit Tagesuebersicht
- Stundenplan nach Wochentagen
- Aufgabenverwaltung mit Faelligkeiten
- Abwesenheiten einreichen und verfolgen
- Ressourcen buchen
- Admin-Bereich fuer Demo- und Organisationsfunktionen
- Offline-Unterstuetzung per Service Worker
- PWA-Grundstruktur mit Manifest

## Technik

- `HTML`
- `CSS`
- `JavaScript`
- `GitHub Pages`
- `Service Worker`
- `Mock API` mit lokalem Demo-Datenbestand

## Projektstruktur

- `index.html` fuer Einstieg und Routing-Shell
- `css/styles.css` fuer das UI und responsive Layout
- `js/pages/` fuer die einzelnen App-Bereiche
- `js/api/mock.js` fuer Demo-Daten und lokale Aktionen
- `sw.js` fuer Offline-Caching

## Ziel

SchulPlan Pro zeigt, wie eine moderne Schulorganisations-App als schnelle Web-Demo aufgebaut werden kann: klar strukturiert, mobil nutzbar und ohne komplexes Backend direkt testbar.
